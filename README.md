Ferramenta de Verificação de Pressão - VC-1
Este é um sistema simples desenvolvido para auxiliar na verificação da pressão da porta da aeronave VC-1.

Como utilizar
Acesse o link do aplicativo.

Insira a temperatura ambiente atual em graus Celsius (°C).

O sistema exibirá a pressão nominal e o intervalo aceitável (±5%) para aquela temperatura.

Objetivo
Agilizar a conferência técnica e garantir que a pressão esteja dentro dos limites operacionais estabelecidos.
